3. Images — Lazy Load + WebP

Add loading="lazy" to all <img> tags

Every <img> in TutorshipSection, TTM, StudySucceed, Instructors, etc. gets loading="lazy" and decoding="async". Browser only downloads images when they're near the viewport.

Install vite-plugin-imagemin for WebP

Adds a build step that converts all .jpg/.jpeg/.png assets to WebP automatically at build time. No manual work needed. 30–50% smaller image payloads.

4. Hero Slideshow — Fix GPU Performance

Before — causes full repaint 
style={{
  backgroundImage: `url(${images[currentIndex]})`,
  transition: 'background-image 1s ease-in'
  // ⚠ CSS can't transition background-image
  // Triggers repaint on every slide change
}}
After — GPU-accelerated opacity crossfade 
// Stack all images, toggle opacity
{images.map((img, i) => (
  <img
    key={i}
    src={img}
    className={`absolute inset-0 w-full h-full
      object-cover transition-opacity duration-1000
      ${i === currentIndex ? 'opacity-100' : 'opacity-0'}`}
    loading={i === 0 ? 'eager' : 'lazy'}
  />
))}
